# Changelog

## 1.0.0 - Unreleased

- OIDC Authorization Code Flow with PKCE

- OidcService for token revocation and SSO logout
- Configurable routes, middleware, and rate limiting
- Migration for `oidc_sub` and `auth_server_refresh_token` fields
